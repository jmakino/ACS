Le projet Kali  est conçu dans l'optique de la construction d'un laboratoire pour les systèmes stellaires denses, tels que les amas stellaires, les régions de formation d'étoiles et les noyaux de galaxies contenant un ou plusieurs trous noirs. La présentation est centrée sur les dialogues entre deux astrophysiciens en train d'écrire le code de Kali à partir de rien.
